# 管理后台 - 独立版本

这是一个独立的管理员后台界面，与用户前端完全分离。

## 启动方式

双击运行 `启动管理后台.bat`

## 访问地址

- **管理后台**: http://localhost:5001
- **登录页面**: http://localhost:5001/login.html
- **仪表板**: http://localhost:5001/dashboard.html

## 默认管理员账号

- **用户名**: admin
- **密码**: 123456

## 端口说明

- **5001** - 管理后台服务（独立）
- **5000** - 用户前端服务
- **8080** - 后端API服务

## 特点

1. ✅ 完全独立，不影响用户前端
2. ✅ 简单的HTML/JS实现，易于维护
3. ✅ 直接调用后端API，无需代理
4. ✅ 自动检查登录状态和管理员权限
5. ✅ 显示实时统计数据

## 注意事项

- 确保后端服务（8080端口）已启动
- 只有ADMIN或SUPER_ADMIN角色才能登录
- Token保存在localStorage中
- 登录过期会自动跳转到登录页

## 文件说明

- `login.html` - 登录页面
- `dashboard.html` - 管理后台主页
- `server.js` - Node.js服务器
- `启动管理后台.bat` - 启动脚本
